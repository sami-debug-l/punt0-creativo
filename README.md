# Punto Creativo — Web (estática)

Sitio web estático para **Punto Creativo** (imprenta en Yapacaní), con paleta **naranja + blanco**.

## Cómo abrir
- Haz doble clic en `index.html`, o
- En VS Code/Cursor usa una extensión tipo “Live Server”.

## Dónde editar rápido
- **Páginas**: `index.html`, `servicios.html`, `trabajos.html`, `opiniones.html`, `contacto.html`
- **Colores/estilos**: `styles.css`
- **WhatsApp** (número y mensaje): `main.js`

En `main.js` cambia:
- `whatsappNumber`: solo dígitos con código país. Ej: `"59171234567"`
- `whatsappMessage`: mensaje por defecto para la cotización.

## Pendientes recomendados (opcional)
- Reemplazar “PC” por tu **logo** real (archivo en `assets/`).
- Agregar fotos reales en “Trabajos recientes”.
- Insertar mapa real (Google Maps) con el enlace o ubicación exacta.
